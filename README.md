# 科学遐思精灵 (Inspiration Storage Genie)

## 项目简介

一款基于 **Tkinter 桌面端**的异步灵感收集与 AI 深度研究工具。允许用户碎片化记录科学遐想，一键触发 DeepSeek AI 分析，AI 生成过程不阻塞用户的实时创作流。

**核心亮点：**
- 🖥️ 桌面端 Tkinter 双栏界面，无需浏览器
- 🔌 零第三方依赖（仅用 Python 内置库）
- 🤖 接入 DeepSeek API，生成专业科研报告
- ⚡ 异步不阻塞，发送后可继续输入新想法
- ⚙️ 图形化设置界面，随时更换 API Key

## 快速开始

```bash
# 1. 进入项目目录
cd 科学遐思精灵

# 2. 直接运行（无需安装任何库）
python main.py
```

系统会自动使用内置默认 Key 连接 DeepSeek API，启动即用。

## 功能

| 功能 | 说明 |
|------|------|
| **暂存累积** | 多次输入灵感碎片，Ctrl+Enter 快捷暂存 |
| **AI 分析** | 一键发送给 DeepSeek，生成5章节科研报告 |
| **异步不阻塞** | 生成过程中可继续输入新想法 |
| **报告库** | 所有报告自动保存，双击查看全文 |
| **设置界面** | ⚙ 按钮 → 随时更换 API Key |
| **零依赖** | 桌面版全部使用 Python 内置库，无需 pip install |

## API Key 管理

- 程序内置默认 Key，首次运行自动写入 `.env`
- 点击标题栏右上角 **⚙ 设置** → 输入新 Key → 保存
- 点「恢复默认」一键回到初始 Key

## 运行测试

python -m unittest discover 科学遐思精灵/tests -v
