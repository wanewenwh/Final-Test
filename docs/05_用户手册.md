# 05 用户手册

## 5.1 系统概述

**科学遐思精灵**是一款基于 Tkinter 桌面端的科学灵感收集与 AI 分析工具。它允许你：

- 随时记录科学灵感碎片
- 多次暂存累积完整想法
- 一键调用 DeepSeek AI 生成结构化科研报告
- AI 生成过程中可继续记录新想法，无需等待
- 图形化设置界面，随时更换 API Key

## 5.2 安装与运行

### 系统要求

- Python 3.9+
- 操作系统：Windows / macOS / Linux
- 无需安装任何第三方库（使用 Python 内置模块）

### 运行方式

```bash
# 进入项目目录
cd 科学遐思精灵

# 直接双击 main.py 或运行：
python main.py
```

系统会自动加载内置默认 Key 连接 DeepSeek API，启动即用。

## 5.4 操作指南

### 场景：记录一个科学遐想并获取 AI 分析

**步骤 1：输入灵感碎片**

在左侧文本框中输入你的第一个想法：

```
我突然想到，如果量子纠缠可以用来传递信息，
是不是意味着我们可以实现超光速通信？
```

**步骤 2：暂存当前碎片**

点击「📌 暂存此句」按钮（或按 `Ctrl+Enter`），文本被追加到下方的暂存区。

**步骤 3：继续补充想法**

输入更多相关想法，再次暂存：

```
但根据相对论，信息的传递不能超过光速，
那量子纠缠在通信中的角色到底是什么？
```

暂存区显示：
```
[1] 我突然想到，如果量子纠缠可以用来传递信息...
[2] 但根据相对论，信息的传递不能超过光速...
```

**步骤 4：发送给 AI 分析**

确认想法完整后，点击「🚀 发送给 AI 生成报告」按钮。

此时：
- ✅ 暂存区自动清空（你可以继续输入新想法）
- ✅ 状态栏显示「⏳ 报告生成中，请稍后...」
- ✅ AI 在后台工作，你不必等待

**步骤 5：查看生成的报告**

等待约 5-15 秒后：
- 右侧「报告库」自动刷新
- 双击新出现的报告条目（或选中后点「查看选中报告」）
- 弹窗显示 AI 生成的结构化报告

报告内容示例：
```
【科学遐思提问】
我突然想到，如果量子纠缠可以用来传递信息...

========================================

【AI 研究报告】

核心假说
量子纠缠能否用于超光速信息传递...

理论背景
根据量子力学非定域性...

可行性分析
挑战：量子不可克隆定理...

潜在价值
若成立将彻底改变通信技术...

下一步建议
1. 研究量子隐形传态最新进展
...
```

## 5.5 设置 API Key

1. 点击标题栏右上角 **⚙ 设置** 按钮
2. 在弹出的对话框中输入你的 DeepSeek API Key
3. 点击「保存」立即生效
4. 如需恢复默认 Key，点击「恢复默认」

## 5.7 报告文件说明

所有生成的报告保存在项目根目录的 `reports/` 文件夹中。

- 文件命名：`科学遐思_YYYYMMDD_HHMMSS_ns.txt`
- 内容格式：
  ```
  【科学遐思提问】

  【AI 研究报告】
  {DeepSeek 返回的结构化文本}
  ```

## 5.8 常见问题 FAQ

### Q: 需要安装什么库吗？
完全不需要。桌面版全部使用 Python 内置模块（tkinter、urllib、json、threading 等），双击 `main.py` 即可运行。

### Q: 没有 DeepSeek API Key 能用吗？
系统内置了默认 Key，启动即用。如果 Key 失效，点击 ⚙ 设置按钮输入你自己的 Key。

### Q: 如何获取自己的 DeepSeek API Key？
访问 https://platform.deepseek.com/ 注册账号，在 API Keys 页面创建。

### Q: 报告生成需要多久？
通常 5-15 秒。生成过程中你可继续输入新想法，无需等待。

### Q: 生成失败怎么办？
常见原因：API Key 无效、网络超时。点击 ⚙ 设置检查 Key 是否正确，或恢复默认 Key。

### Q: 如何查看之前生成的报告？
在右侧报告列表中双击报告条目即可查看。所有报告也保存在 `reports/` 文件夹中，可用任何文本编辑器打开。

### Q: 报告文件太多怎么管理？
`reports/` 目录下的文件可按文件名中的时间戳排序，也可以手动归档或删除。
